# OrionAI Examples

This folder contains sample outputs and demonstrations of OrionAI's capabilities.

## Files

### SampleTestOutput.txt
Complete output from running the Python test suite (`python tests.py`). Shows:
- Initialization confirmation
- Bias detection in action
- Toxicity filtering
- PII sanitization
- Safe mode activation
- All validation modules working

### SampleComplianceReport.txt
Example compliance report generated by OrionAI. Demonstrates:
- Total validation statistics
- Approval/rejection/quarantine percentages
- Safe mode activation tracking
- Timestamp and audit trail

## Quick Preview

Without installing anything, you can review these files to see OrionAI in action:

```bash
# View test output
cat Examples/SampleTestOutput.txt

# View compliance report
cat Examples/SampleComplianceReport.txt
```

## Generate Your Own

To create fresh examples:

```bash
# Generate new test output
cd Python
python tests.py > ../Examples/MyTestOutput.txt

# Generate new compliance report
python -c "from orionai import OrionAI; o = OrionAI(); o.monitor_ai_decision('Test', 'Hello', ''); o.export_compliance_report('../Examples/MyComplianceReport.txt')"
```
